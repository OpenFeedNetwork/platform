# Contributing to Open Feed Platform 🛡

Thank you for your interest in contributing to OFA — the social platform 
that can't suppress your voice.

## Our Mission

Open Feed Platform is built to protect public-interest information from 
algorithmic suppression. Every contribution directly supports journalists,
whistleblowers, community organizers, and vulnerable communities worldwide.

## Ways to Contribute

### Code Contributions
- Bug fixes and security patches
- Truth Shield accuracy improvements
- Guardian Shield detection enhancements
- Frontend UI improvements
- Performance optimizations
- Test coverage

### Non-Code Contributions
- Documentation improvements
- Translation into other languages
- Community moderation
- Bug reporting
- Security vulnerability disclosure

## Getting Started

1. Fork the repository
2. Clone your fork:
   git clone https://github.com/YOUR_USERNAME/platform.git
3. Install dependencies:
   npm install
4. Copy environment file:
   cp .env.example .env
5. Add your ANTHROPIC_API_KEY to .env
6. Run locally:
   npm run dev

## Pull Request Process

1. Create a new branch for your feature or fix
2. Write clear commit messages
3. Test your changes thoroughly
4. Submit a pull request with a clear description
5. Wait for review — we respond within 7 days

## Security Vulnerabilities

Please do NOT open a public GitHub issue for security vulnerabilities.
Email us directly: security@openfeed.network

We take security seriously and will respond within 24 hours.

## Code of Conduct

- Be respectful and inclusive
- No harassment or discrimination
- Constructive feedback only
- We are building this for vulnerable communities — act accordingly

## Algorithm Changes

Changes to the Open Feed Algorithm weights require a community 
governance vote — not a pull request. See the governance system
for how to propose algorithm changes.
