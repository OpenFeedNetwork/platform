Architecture & Security
BUILT TO STAY CLOSED

The open feed earns trust through transparency. The safety suite earns trust through impenetrability. Both principles are non-negotiable.

🏗
Encrypted API layer
All CandorShield modules operate on an isolated, encrypted cloud architecture separate from the public Candor Reach feed. Detection logic never touches the open repository.
🔁
Hash-based data pipeline
Content flows from the Reach pipeline into the Shield environment via encrypted hash tokens — not raw content. No post text ever leaves your infrastructure unencrypted.
🚫
Zero public exposure
Detection thresholds, scoring weights, and signature databases are never published. No open-source repository. No public documentation of internal logic. No bypass surface.
🌐
Multi-region deployment
Infrastructure deployed across iad, lax, and ams regions via Fly.io. Geographic redundancy ensures uptime commitments are maintained even during regional incidents.
📜
Compliance-ready
CandorShield modules are architected for HIPAA, COPPA, FedRAMP, and SOC 2 compliance paths depending on module. Compliance documentation available under NDA.
🤝
Access by agreement only
No self-serve access. All CandorShield modules are deployed under signed agreements with use case review. Government and enterprise procurement processes supported.
