## What Makes OFA Different

| Standard Platforms | Open Feed Platform |
|---|---|
| Black-box suppression | Transparent AI review |
| No appeal process | 30-day community appeals |
| Ads boosted by default | Ads penalized and disclosed |
| Auto-deletion | Context labels only — never deleted |
| No audit trail | Every decision on IPFS forever |

## Core Systems

- 🛡 **Truth Shield** — Claude AI analyzes every suppression flag before any demotion
- 👁 **Guardian Shield** — 7-layer child protection, COPPA/GDPR-K compliant
- 📡 **Open Feed Algorithm** — Transparent ranking, community-governed weights
- 🔐 **Zero-Knowledge Age Verification** — Prove 18+ without revealing identity
- ◆ **IPFS + Arweave** — Permanent decentralized storage, censorship-resistant
- 🗳 **On-Chain Governance** — Algorithm changes require community votes

## Quick Start

```bash
git clone https://github.com/OpenFeedNetwork/platform.git
cd platform
npm install
cp .env.example .env
# Add your ANTHROPIC_API_KEY to .env
./scripts/deploy.sh local
```
