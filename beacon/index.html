CANDOR BEACON · MENTAL HEALTH MONITOR

Real-time suicide prevention for the general population

Candor Beacon continuously monitors the open feed for behavioral and linguistic signals associated with suicidal ideation in the general population — enabling early intervention before crisis escalates.

View API Docs Contact Us

How Beacon Works

Three-stage pipeline from signal detection to response coordination

🔍
SIGNAL DETECTION
Natural language processing scans posts in real time for linguistic patterns, behavioral signals, and contextual markers associated with suicidal ideation.
⚖️
RISK SCORING
Each signal is scored on a 0–100 risk scale using a multi-factor model trained on clinical literature and validated behavioral datasets.
🤝
RESPONSE PROTOCOL
High-risk signals trigger configurable response protocols — from in-app resource surfacing to webhook notifications for crisis intervention teams.

Detection Signals

Beacon monitors across multiple signal dimensions simultaneously

PRIMARY LINGUISTIC SIGNALS
Direct expressions of suicidal ideation, farewell statements, expressions of hopelessness, statements of being a burden, references to method or means, and goodbye language patterns.
BEHAVIORAL SIGNALS
Sudden social withdrawal patterns, unusual posting frequency changes, giving away possessions or accounts, expressions of finality, and isolation language.
CONTEXTUAL SIGNALS
Recent loss events, crisis anniversaries, seasonal patterns, community context, and engagement with high-risk content or communities.

API Reference

Integrate Beacon into your platform

// Analyze a post for mental health signals
POST /api/v1/beacon/analyze

{
  "post_id": "uuid",
  "content": "post content here",
  "user_id": "anonymous-hash",
  "context": {
    "prior_posts": 3,
    "account_age_days": 180
  }
}

// Response
{
  "risk_score": 0.82,
  "risk_level": "high",
  "signals": ["farewell_language", "hopelessness"],
  "recommended_action": "crisis_resources",
  "resources": {
    "hotline": "988",
    "text": "Text HOME to 741741"
  }
}
